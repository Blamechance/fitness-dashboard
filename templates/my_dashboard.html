Analysing Training Data:

The Strength Index (SI) is a custom formula designed to measure the strength feat of an exercise set.

It takes into account context of athlete bodyweight, as well as reps carried out. This allows athletes to tangibly measure progress in strength, if the user achieves the same lift at a lower body weight. For recorded sets where the user also logged their body weight ( +/-3 days for a 7 day window of relevance), a strength index is calculated.

The formula is as follows:

[ Reps x weight ] / [ bodyweight(kg) x Rep Range Factor* ]

*Rep Range factors will be utilised based on the following scaling model:

  • 1-2 reps = 3.3
  • 3-6 reps = 1.4
  • 7-10 reps = 1 (normal)
  • 11-15 reps = 0.8
  • 16-30 reps = None (don't calculate due to obvious skewing)
