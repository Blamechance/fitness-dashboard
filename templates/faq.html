How to use this tool:

In the FitNotes app:

Open the app and click on the 3 dots on the top-right and click "Settings". Then:

  1. Make sure at the very top, the Unit system is set to "Metric".
  2. Scroll down half-way until you see the "Spreadsheet Export" option.
  3. In the pop-up menu, uncheck "Include Timestamp in file name". Support for Comment data is provided, so feel free to leave that checked.
  4. Click on "Save export" to save to your device.
  5. Repeat, but this time after clicking "spreadsheet Export", check "Body Tracker Data" to export weight data.

In Fitness Dashboard Website:

Navigate to the "Monthly Check-in" page and upload your respective files!

How is the data processed?

Analysing Training Data:

The Strength Index (SI) is a custom formula designed to measure the strength feat of an exercise set, reagardless of fluctuating context (bodyweight, reps executed etc).

A Strength Index is calculated for recorded sets where weight data is available -- this is a custom formula designed to measure the strength feat of an exercise set.

* This weight value is the average of all available weight data, within a 7 day proximity window (3 days on either side + the date).

Strength Index Formula:

[ Reps x weight ] / [ bodyweight(kg) x RR Factor* ]

*Rep Range (RR) factors will be utilised based on the following scaling model:
Reps Range Scaling Factor
1-2 reps 3.3
3-6 reps 1.4
7-10 reps 1 (normal)
11-15 reps 0.8
16+ reps None (don't calculate due to obvious skewing)

Analysing Weight Data:

Weight chart values are averages of all entries between each other.

Both fitnotes files are cross-referenced to provide the weight data of that date, next to any related exercises in the Training Data table.

Only metric system (kg) is supported.
